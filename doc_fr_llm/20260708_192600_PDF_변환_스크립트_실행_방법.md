# make_pdf_with_file.py 실행 방법 가이드

이 문서는 Jupyter Notebook (`.ipynb`) 또는 Python 스크립트 (`.py`) 파일을 PDF로 변환해 주는 자동화 도구인 `make_pdf_with_file.py`를 로컬 터미널 환경에서 실행하기 위한 사전 설정 및 명령어 사용법을 정리한 가이드입니다.

---

## 1. 사전 준비 (Environment Settings)

### ① Conda 가상환경 활성화
이 프로젝트의 주요 학습 및 컴파일 패키지가 설치된 `logsentinel` 가상환경을 활성화합니다.
```bash
conda activate logsentinel
```

### ② 의존성 패키지 설치
`requirements.txt`에 명세된 jupyter, nbconvert, playwright 등의 패키지를 최신화합니다.
```bash
pip install -r requirements.txt
```

### ③ Playwright 브라우저 드라이버 설치 (필수)
`make_pdf_with_file.py`는 내부적으로 Headless Chromium 브라우저를 백그라운드에서 띄워 화면을 PDF로 굽는 `WebPDFExporter`를 사용합니다. 따라서 반드시 로컬 크로미움 브라우저 드라이버를 설치해야 합니다.
```bash
playwright install
```

---

## 2. 명령어 사용법 (CLI Command)

### ① 기본 실행 (원본 파일명과 동일한 PDF 생성)
변환 대상 파일의 경로만 인자로 넘겨주면, 원본 파일이 있는 동일한 위치에 확장자만 `.pdf`로 변경되어 저장됩니다.
```bash
python make_pdf_with_file.py [대상_파일_경로]
```
* **Jupyter Notebook 변환 예시**:
  ```bash
  python make_pdf_with_file.py middle_high_school_math/01_middle_1/03_algebraic_expressions/01_03_algebraic_expressions.ipynb
  # 결과물: middle_high_school_math/01_middle_1/03_algebraic_expressions/01_03_algebraic_expressions.pdf 생성
  ```
* **Python 스크립트 변환 예시**:
  `# %%` 주석을 기준으로 임시 주피터 노트북 셀을 자동 빌드하여 PDF로 생성합니다.
  ```bash
  python make_pdf_with_file.py run_simulation.py
  # 결과물: run_simulation.pdf 생성
  ```

### ② 출력 경로 직접 지정 실행 (`-o` 또는 `--output` 옵션)
원하는 경로와 파일명으로 PDF를 저장하고 싶을 때 `-o` 옵션을 추가합니다.
```bash
python make_pdf_with_file.py [대상_파일_경로] -o [원하는_PDF_저장_경로]
```
* **실행 예시**:
  ```bash
  python make_pdf_with_file.py middle_high_school_math/01_middle_1/03_algebraic_expressions/01_03_algebraic_expressions.ipynb -o documents/algebra_study.pdf
  ```
