# 2026-07-04 factor_tree.py 실행 검증 및 경로 개선 기록

이 문서는 소인수분해 트리 시각화 스크립트([factor_tree.py](file:///Users/joshuapark/Desktop/Dev/soromiso/circlemath/middle_high_school_math/01_middle_1/01_prime_factorization/factor_tree.py))의 정상 실행 여부를 확인하고, 실행 경로와 무관하게 동작하도록 소스 코드를 개선한 내역을 담고 있습니다.

## 1. 문제 파악
- 기존 소스 코드에서는 시각화 이미지의 저장 경로(`output_path`)가 `01_prime_factorization/factor_tree_{number}.png`와 같은 상대 경로로 하드코딩되어 있었습니다.
- 이로 인해 프로젝트 루트 디렉토리([circlemath](file:///Users/joshuapark/Desktop/Dev/soromiso/circlemath)) 등에서 스크립트를 실행할 때, 기준 디렉토리 아래에 `01_prime_factorization` 폴더가 존재하지 않아 `FileNotFoundError: [Errno 2] No such file or directory` 에러가 발생하며 스크립트 실행이 중단되는 문제가 확인되었습니다.

## 2. 해결 방안 및 수정 내역
- 실행 경로(CWD)와 상관없이 스크립트가 위치한 디렉토리에 정상적으로 결과 이미지가 생성되도록 절대 경로 조회 로직을 적용했습니다.
- **수정 상세**:
  - `os` 모듈을 임포트하였습니다.
  - `output_path` 설정을 다음과 같이 동적 절대 경로로 계산하도록 수정하였습니다.
    ```python
    current_dir = os.path.dirname(os.path.abspath(__file__))
    output_path = os.path.join(current_dir, f"factor_tree_{number}.png")
    ```

## 3. 실행 검증 결과
- Conda 가상환경 `logsentinel` 환경을 사용하여 검증을 수행하였습니다.
  ```bash
  conda run -n logsentinel python middle_high_school_math/01_middle_1/01_prime_factorization/factor_tree.py
  ```
- 실행 후 [factor_tree_120.png](file:///Users/joshuapark/Desktop/Dev/soromiso/circlemath/middle_high_school_math/01_middle_1/01_prime_factorization/factor_tree_120.png) 이미지가 정상적으로 새로 생성 및 갱신된 것을 확인하였습니다. (성공적으로 시각화 및 이미지 저장이 완주됨)
