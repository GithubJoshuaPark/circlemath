1. **노트북 파일 물리적 복구**:

2. **`circle` 가상환경 커널 스펙 등록**:
   - 다음 명령어를 실행하여 `circle` 환경의 커널 스펙을 시스템에 명시적으로 설치했습니다.
     `conda run -n circle python -m ipykernel install --user --name circle --display-name "Python 3 (circle)"`
   - 이제 IDE 우측 상단 커널 선택 메뉴에서 `Python 3 (circle)`을 직접 지정해 안전하게 노트북을 작동시킬 수 있습니다.
