## 2. README.md 구성 핵심 요약
`README.md`는 중고등학교 수학 수준에서 다룰 수 있는 시각화 주제들을 체계적으로 분류하고, 프로젝트명 `circles`에 초점을 맞추어 기획 및 기술 가이드를 포함해 작성되었습니다.

### 주요 포함 내용
1. **기획 의도**: 수식의 움직임을 정적/동적 시각화하여 수학의 직관적인 이해를 돕기 위함.
2. **주요 시각화 주제**:
   - **삼각함수와 단위원**: 단위원 위 동경의 위치와 삼각함수 그래프 연동 애니메이션.
   - **원주율($\pi$) 추정**: 아르키메데스의 다각형 법, 몬테카를로 기법 시각화.
   - **원추곡선**: 원, 타원, 포물선, 쌍곡선의 3D 및 2D 단면 및 기하 정의 자취 시각화.
   - **미적분과 극한**: 할선에서 접선으로의 변화, 구분구적법 사각형 분할 시각화.
   - **에피사이클로이드 & 푸리에 급수**: 회전하는 원들을 이용한 궤적 시각화.
3. **기술 스택**: NumPy, Matplotlib, SymPy, Manim, Jupyter Notebook.
4. **설치 및 디렉토리 구조 제안**: 프로젝트의 실행 방법 및 확장 가능한 파일 구조 가이드 제시.
