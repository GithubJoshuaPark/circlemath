### ① 라이브러리 임포트 (가져오기)
```python
import matplotlib.pyplot as plt
import numpy as np
from fractions import Fraction
import platform
```
- **`import matplotlib.pyplot as plt`**: 파이썬에서 그래프나 도표를 그릴 때 사용하는 대표적인 라이브러리입니다. 이름이 길기 때문에 앞으로 코드 내에서 편리하게 부르기 위해 `plt`라는 별칭(alias)을 붙였습니다.
- **`import numpy as np`**: 수치 연산, 수학 함수, 배열(Array) 처리를 효율적으로 하기 위한 라이브러리입니다. 루트 계산($\sqrt{2}$) 등을 위해 사용하며 `np`라는 별칭으로 부릅니다.
- **`from fractions import Fraction`**: 파이썬 내장 라이브러리 중에서 분수(Fraction)를 다루는 도구입니다. 이 클래스는 분자(numerator)와 분모(denominator)를 기반으로 작동하며, 분수를 넣으면 자동으로 **기약분수**로 약분해 주는 편리한 성질을 가지고 있습니다.
- **`import platform`**: 코드가 실행되는 운영체제(Windows, Mac 등)의 정보를 알아내기 위한 라이브러리입니다.

- **`ax.set_facecolor('#0D1117')`**: 실제 그래프가 그려지는 내부 플롯 영역의 배경색도 어두운 테마에 맞춰 칠해줍니다.

### ④ 기약분수 루프 및 데이터 추출
```python
    rationals_x = []
    rationals_y = []
    
    for q in range(1, max_denominator + 1):
        for p in range(0, q + 1):
            # 기약분수만 필터링하여 중복 점 제거
            frac = Fraction(p, q)
            if frac.denominator == q:
                val = p / q
                rationals_x.append(val)
                rationals_y.append(q)
```
- **`rationals_x = []`, `rationals_y = []`**: 시각화할 점들의 X좌표(유리수의 실수 값)와 Y좌표(분모 $q$)를 담아둘 빈 리스트(주머니)입니다.
- **`for q in range(1, max_denominator + 1):`**: 분모 $q$를 1부터 `max_denominator`(여기서는 20)까지 1씩 증가시키며 바깥쪽 반복문을 돕니다.
- **`for p in range(0, q + 1):`**: 각 분모 $q$에 대해, 분자 $p$를 0부터 $q$까지 1씩 증가시키는 안쪽 반복문을 돕니다. 즉, $0/q$부터 $q/q$ ($1$) 사이의 모든 분수를 탐색합니다.
- **`frac = Fraction(p, q)`**: 파이썬의 `Fraction` 클래스를 호출해 분수를 생성합니다. 예를 들어 $2/4$를 생성하면, 파이썬 내부에서 자동으로 약분하여 `1/2`로 저장합니다.
- **`if frac.denominator == q:`**: **(가장 핵심적인 필터링)** 만약 약분된 최종 기약분수의 분모(`frac.denominator`)가 원래 분모인 $q$와 똑같다면, 이 분수는 약분되지 않는 고유한 분수라는 의미입니다. 
  - 예를 들어 $2/4$는 약분되어 분모가 `2`가 되므로 $q$($4$)와 달라져 이 `if` 조건문을 통과하지 못합니다. (이미 $1/2$ 차례에서 점을 찍었기 때문에 중복을 제거하기 위함)
- **`val = p / q`**: 분수를 실수 소수점으로 환산합니다. (예: $1/2$ ➡️ $0.5$)
- **`rationals_x.append(val)`**, **`rationals_y.append(q)`**: 필터링을 통과한 소수 값을 $x$좌표에 넣고, 그때의 분모 $q$를 $y$좌표에 추가합니다.

### ⑤ 산점도(Scatter) 그리기와 점 크기 동적 조절
```python
    # 유리수 분포 점으로 표현 (분모가 작을수록 더 크고 뚜렷하게)
    sizes = [150 / y for y in rationals_y]
    ax.scatter(rationals_x, rationals_y, color='#0B79D0', s=sizes, alpha=0.7, zorder=2, label='유리수 (p/q)')
```
- **`sizes = [150 / y for y in rationals_y]`**: 리스트 컴프리헨션(List Comprehension) 문법을 사용해 점들의 크기 리스트를 동적으로 만듭니다. Y좌표인 분모 $y$가 작을수록(예: 1, 2) 나누는 값이 작아지므로 점의 크기(`s`)가 커지고, 분모가 커질수록(예: 20) 점의 크기가 아주 미세해집니다. 이를 통해 수의 기하학적 위계를 아름답게 나타냅니다.
- **`ax.scatter(...)`**: 산점도 그래프를 그리는 핵심 함수입니다.
  - `color='#0B79D0'`: 은은하고 신뢰감을 주는 파란색으로 칠합니다.
  - `s=sizes`: 방금 계산한 점의 크기 리스트를 적용합니다.
  - `alpha=0.7`: 살짝 반투명하게 하여 촘촘히 겹치는 구간에서 유리수들의 조밀성을 부드럽게 감상할 수 있게 돕습니다.
  - `zorder=2`: 레이어의 순서(높이)를 지정합니다. 숫자가 클수록 다른 선들 위에 얹혀 그려집니다.

### ⑥ 무리수 표시 (axvline)
```python
    # 대표적인 무리수 1/sqrt(2) 표시
    irrational_val = 1 / np.sqrt(2)
    ax.axvline(irrational_val, color='#ff5a5f', linestyle='--', linewidth=2, zorder=3,
               label=r'무리수 $1/\sqrt{2}$ ($\approx$ ' + f'{irrational_val:.4f})')
```
- **`irrational_val = 1 / np.sqrt(2)`**: 대표적인 무리수인 $\frac{1}{\sqrt{2}}$ ($\approx 0.7071$) 값을 구합니다.
- **`ax.axvline(...)`**: 차트의 $x$축 특정 값 위치에 수직선(Vertical Line)을 그리는 특수 함수입니다.
  - `color='#ff5a5f'`: 강렬한 Stark Red 컬러를 사용해 시선을 끕니다.
  - `linestyle='--'`: 점선으로 표현합니다.
  - `label=...`: 범례에 표시할 라벨입니다. 이스케이프 문자 충돌을 막기 위해 raw string(`r'...'`)과 f-string을 결합하여 수학 수식 $\approx$과 소수점 넷째 자리($0.7071$)를 출력하도록 구성했습니다.

### ⑦ 축 레이아웃 및 범례 꾸미기
```python
    # 축 및 꾸미기
    ax.set_title(f"유리수의 조밀성과 무리수의 빈틈 (최대 분모: {max_denominator})", color='white', fontsize=15, pad=15)
    ax.set_xlabel("수직선 (0 ~ 1 구간)", color='white', fontsize=12)
    ax.set_ylabel("분모 (Denominator, q)", color='white', fontsize=12)
    
    ax.tick_params(colors='white')
    ax.spines['bottom'].set_color('#8B949E')
    ax.spines['left'].set_color('#8B949E')
    ax.spines['top'].set_visible(False)
    ax.spines['right'].set_visible(False)
    
    ax.grid(color='#30363D', linestyle=':', alpha=0.5)
    ax.set_ylim(max_denominator + 1, 0)  # 분모가 작을수록 위에 그리도록 Y축 반전
    
    legend = ax.legend(loc='lower right', facecolor='#161B22', edgecolor='#30363D', labelcolor='white')
    plt.tight_layout()
    plt.show()
```
- **`ax.set_title()`, `ax.set_xlabel()`, `ax.set_ylabel()`**: 제목과 각 축의 이름, 폰트 크기, 패딩(여백)을 설정합니다.
- **`ax.tick_params(colors='white')`**: 축 눈금 수치들의 색상을 흰색으로 만들어 시인성을 극대화합니다.
- **`ax.spines[...]`**: 그래프 테두리(Spine) 선들의 색상을 조절하거나 보이지 않게 만듭니다. 위쪽과 오른쪽 테두리를 숨겨 화면을 트이게 구성했습니다.
- **`ax.grid(...)`**: 플롯 뒤쪽에 모눈망(격자망)을 옅은 회색 점선으로 깔아줍니다.
- **`ax.set_ylim(max_denominator + 1, 0)`**: **(기하학적 중요 포인트)** $y$축의 범위를 일반적인 `0`에서 `20`이 아닌, `21`에서 `0`으로 뒤집었습니다. 이렇게 하면 분모가 1인 수($1.0$)와 분모가 2인 수($0.5$)가 차트 최상단에 큰 점으로 먼저 넓게 퍼지고, 아래로 내려갈수록(분모가 20에 가까워질수록) 미세한 작은 점들이 지극히 조밀하게 뭉치는 **아름다운 프랙탈 구조의 계층**이 뒤집혀 안정적으로 드러나게 됩니다.
- **`ax.legend()`**: 범례 박스를 어두운 색 배경과 흰색 글씨로 예쁘게 연출합니다.
- **`plt.tight_layout()`**: 그래프 여백을 자동으로 알맞게 조정하여 축의 라벨이 잘리는 현상을 막습니다.
- **`plt.show()`**: 생성된 그림을 최종적으로 화면에 렌더링하여 보여줍니다.
