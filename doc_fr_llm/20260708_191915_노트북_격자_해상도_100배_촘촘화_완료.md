# 2026-07-08 데카르트 페인터 격자 해상도 100배 촘촘화 패치 및 최적화 완료

Joshua님의 요청에 따라 `01_04_cartesian_painter_trajectory_tracker.ipynb` 파일의 세 번째(수식 데이터 생성 및 플로팅), 네 번째(슬라이더 interact 실행) 코드 셀 내부 격자 및 캔버스 해상도를 100배 더 정밀하게 조밀화하였습니다.

## 1. 개선 내역
- **격자 조밀화**:
  - 기존 1 간격으로 구성된 정수 격자(`grid_x`, `grid_y`)를 `0.01` 간격으로 고해상도 촘촘화 완료.
  - 가로/세로 전체 픽셀 해상도가 $21 \times 21$에서 $2001 \times 2001$로 100배 확장되었습니다.
- **Numpy 벡터 연산 도입 (실시간 속도 최적화)**:
  - 100배 조밀해진 해상도에서 400만 번의 파이썬 중첩 `for` 루프를 순회할 경우 발생하는 심각한 실시간 UI 렉 현상을 극복하기 위해, **Numpy meshgrid 기반의 벡터 행렬 연산**으로 데이터 생성 방식을 변경했습니다.
  - 이를 통해 연산 시간을 단 0.03초 이내로 단축하여 실시간 슬라이더 이동 시 즉시 매끄럽게 플로팅되도록 보완했습니다.
- **안티 브루탈리즘 스타일링 보간**:
  - 해상도 촘촘함에 맞추어 오차 필터링 임계치를 `0.05~0.065`로 축소 조정하여 선명하고 일정한 두께의 그래프 선을 구현했습니다.
  - 슬라이더 GUI의 미세 조정을 극대화하고자 조절 스텝(`step`)을 `0.2`에서 `0.05`로 촘촘화하였습니다.
