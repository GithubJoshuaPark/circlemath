# 2026-07-08 PDF 변환 스크립트 실행 경로 오류 분석 및 디버깅

Jupyter Notebook을 PDF로 변환하는 스크립트 실행 과정에서 무출력 오류(Exit Code 1)가 발생하여 이를 조사하고 해결책을 마련하였습니다.

## 1. 발생 문제
- 사용자가 아래 명령어를 가동하였으나, 아무런 메시지 없이 실행이 실패(exit 1)함:
  `python make_pdf_with_file.py ./middle_hight_school_math/...`

## 2. 원인 분석
1. **경로 오타 (Path Typo)**:
   - 프로젝트 내 주피터 노트북 경로의 실제 폴더명은 `middle_high_school_math` (high) 입니다.
   - 사용자가 입력한 경로명은 `middle_hight_school_math` (hight) 로 철자 오류가 발생하여 파일이 존재하지 않는 에러가 났습니다.
2. **에러 로깅 누락**:
   - `make_pdf_with_file.py` 스크립트의 메인 예외 처리 블록(`try-except`)에서 에러 상황 시 에러 내용을 출력하지 않고 조용히 `sys.exit(1)`만 가동하여, 오류 원인을 추적하기 어려웠습니다.
