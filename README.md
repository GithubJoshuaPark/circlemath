## 1. 소개 및 기획 의도

청소년 시절 우리는 제한된 시간 안에 문제를 풀어내고 점수를 얻기 위해 수학을 빠르게 소비하곤 했습니다.
공식이 도출된 이면의 거대한 생각의 역사나 기하학적 아름다움은 스쳐 지나갈 수밖에 없었습니다.

본 프로젝트는 50대의 긴 호흡으로 돌아온 Joshua님을 위해 기획되었습니다. 수학 공식이 가지고 있는 **수학적 의미, 실제 생활에서의 적용점, 철학적·종교적 사유의 연결고리**를 천천히 음미하고, **파이썬(Python) 코딩과 시각화**라는 강력한 현대적 도구를 통해 직접 만지고 느끼는 체험적 수학 학습을 지향합니다.

## 4. 주요 시각화 주제

프로젝트의 본령인 기하학 및 삼각함수, 원의 성질과 관련된 대표적인 핵심 시각화 기획입니다.

- **3.1. 삼각함수와 단위원**: 단위원 위 동경의 위치와 삼각함수 그래프 연동 애니메이션
- **3.2. 원주율($\pi$)의 다양한 추정**: 아르키메데스의 다각형 근사, 몬테카를로 난수 추정
- **3.3. 원추곡선 (Conic Sections)**: 원뿔의 단면 변화를 통한 원, 타원, 포물선, 쌍곡선의 동적 시각화
- **3.4. 미적분과 극한**: 할선이 접선으로 수렴하는 미분계수의 한계값, 구분구적법 분할
- **3.5. 에피사이클로이드와 푸리에 급수**: 회전하는 원들의 합성 궤적 및 푸리에 경로 드로잉

## 5. 기술 스택 및 요구 라이브러리

본 시각화 프로젝트는 다음과 같은 파이썬 라이브러리를 활용합니다.

- **NumPy**: 고성능 수치 계산 및 다차원 배열 연산.
- **Matplotlib / Seaborn**: 2D 인터랙티브 그래프 및 실시간 애니메이션 생성.
- **Manim (Community Edition)**: 칠판 스타일의 아름답고 직관적인 수학 교육용 애니메이션 렌더링.
- **SymPy**: 기호 수학(Symbolic Mathematics) 연산을 통한 수식 유도 및 해석적 계산.
- **Jupyter Notebook**: 코드 실행과 시각화 결과를 한눈에 확인하는 인터랙티브 환경.
